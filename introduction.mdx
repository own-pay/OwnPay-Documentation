OwnPay is an enterprise-grade, open-source payment orchestration platform designed for complete data ownership and white-label operations. Instead of relying on centralized payment processors with high markups and restricted data visibility, OwnPay allows you to run your own gateway server, connect 100+ payment networks, and manage multiple isolated merchant brands from a single interface.

---

## Core Features

- **Sovereign White-Labeling**: Host checkout flows on brand-specific custom domains (e.g. `pay.yourbrand.com`) with independent logos, color schemes, and checkout templates.
- **Unified Gateway Routing**: Connect card processors, local mobile wallets (such as bKash and Nagad), and crypto networks under a single unified API.
- **Double-Entry Ledger**: Maintain absolute accounting integrity with a built-in double-entry bookkeeping engine that logs all movements of funds.
- **Self-Hosted Autonomy**: Runs on standard modern PHP environments (PHP 8.3+), supporting shared hosting, VPS, dedicated servers, or Docker containers.
