## Welcome to OwnPay

OwnPay is a self-hosted, open-source payment gateway platform that gives you complete control over payment processing. This documentation covers everything you need to use, integrate with, and extend OwnPay.

### Choose Your Path

**Are you an Administrator?**
→ Start with the [User Guide](/user-guide/) to manage brands, payments, users, and system settings.

**Are you a Developer?**
→ Jump to the [Developer Guide](/developer/) to integrate OwnPay via API, webhooks, or plugins.

**Are you new to OwnPay?**
→ Begin with [Getting Started](/getting-started/) for installation and first steps.

**Want to understand the platform?**
→ Explore [Core Concepts](/core-concepts/) to learn the architecture and fundamental ideas.

---

## Key Features of OwnPay

- **Multi-brand Management** - Support multiple stores, brands, or merchants from one platform
- **Payment Gateway Integration** - Connect to popular payment gateways and payment methods
- **Flexible Permissions** - Role-based access control (RBAC) for secure team collaboration
- **Webhooks & Events** - Real-time event notifications for payment updates
- **Mobile Integration** - SMS capabilities and device pairing for extended features
- **Financial Reporting** - Comprehensive transaction reports and audit logs
- **Plugin System** - Extend OwnPay with custom plugins and hooks
- **API-First Design** - Complete REST API for programmatic access
