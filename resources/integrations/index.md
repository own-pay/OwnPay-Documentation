# Integration Guides

Integrate OwnPay with your application using any of our available SDKs and frameworks.

We provide official SDKs for the most popular languages and frameworks. For other languages or frameworks, use the REST API directly.

## Available SDKs

- **PHP** - [PHP SDK](/resources/integrations/rest-api)
- **Node.js** - [Node.js SDK](/developer/integration/nodejs)
- **Python** - [Python SDK](/resources/integrations/python)
- **Java** - [Java SDK](/resources/integrations/java)
- **Go** - [Go SDK](/resources/integrations/go)
- **Ruby** - [Ruby SDK](/resources/integrations/ruby)
- **JavaScript/TypeScript** - [JavaScript SDK](/resources/integrations/javascript)
- **.NET / C#** - [.NET SDK](/resources/integrations/dotnet)
- **Mobile** - [iOS/Android](/resources/integrations/mobile)

## REST API

For other languages or custom implementations, use the [REST API](/resources/integrations/rest-api) directly.
