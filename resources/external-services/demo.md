## How to Use

### 1. Access the Demo

1. Go to [demo.ownpay.org/login](https://demo.ownpay.org/login)
2. Enter credentials above
3. You're in the demo!

### 2. Explore Features

- Create a payment link
- Process a test payment
- View transaction details
- Generate a report
- Create an invoice
- Add a user with custom role
- Install a plugin
- Configure a test gateway

### 3. Create Test Payments

1. Go to **Payments → Payment Links**
2. Click "Create Payment Link"
3. Enter amount (any amount works)
4. Set customer email
5. Click "Create"
6. Share the generated link

### 4. Test Checkout

1. Click the payment link
2. Test the checkout experience
3. Use test card numbers:
   - **Visa:** 4111 1111 1111 1111
   - **Mastercard:** 5555 5555 5555 4444

4. See the payment process

### 5. Explore Admin Features

- Navigate the admin panel
- Check each section
- View sample data
- Try different features
- Get a feel for the UI

## Common Demo Tasks

### Create a Payment Link

1. **Payments → Payment Links**
2. Click "Create Payment Link"
3. Set amount, customer, description
4. Click "Create"
5. Copy generated link
6. Share or test

### Process a Payment

1. Click payment link
2. Enter test card (see above)
3. Click "Pay Now"
4. See payment success page
5. Check transaction in **Payments → Transactions**

### View Reports

1. **Reports → Financial Reports**
2. Select date range
3. See revenue, transactions, summary
4. Export if needed

### Manage Users

1. **People → Staff**
2. Click "Add Staff"
3. Enter email and name
4. Assign role
5. Send invite

### Configure Gateway

1. **Gateways → Payment Gateways**
2. Click "Add Gateway"
3. Select gateway type
4. Enter test credentials
5. Save and test
