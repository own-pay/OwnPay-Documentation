# Plugin Marketplace

The OwnPay Plugin Marketplace is the official hub for discovering, installing, and publishing plugins and themes for OwnPay.

**Website:** [plugins.ownpay.org](https://plugins.ownpay.org)

## What You'll Find

### Gateway Plugins
Extend OwnPay with new payment methods and processors:
- **Stripe**, **PayPal**, **Razorpay**, **2Checkout** integrations
- **Local payment methods** (bKash, Nagad, GCash, etc.)
- **Cryptocurrency** payment processors
- **Custom gateway** implementations

### Addon Plugins
Enhance OwnPay functionality:
- **Email notifications** and templating
- **SMS integrations** (Twilio, AWS SNS, etc.)
- **Analytics** and reporting tools
- **Accounting software** connectors (QuickBooks, FreshBooks)
- **CRM integrations** (Salesforce, HubSpot)
- **Inventory management** syncing

### Themes
Customize the checkout experience:
- **Minimal** checkout themes
- **Dark mode** designs
- **Custom branding** templates
- **Mobile-optimized** themes
- **Accessibility-first** designs

## How to Use

### Install a Plugin
1. Log in to OwnPay admin panel
2. Go to **System → Plugins**
3. Click **Browse Marketplace**
4. Search or browse plugins
5. Click **Install**
6. Activate the plugin

### Publish Your Plugin
1. Develop your plugin locally
2. Test thoroughly with OwnPay
3. Create plugin.json manifest
4. Go to [plugins.ownpay.org/contact](https://plugins.ownpay.org/contact.php)
5. Submit for review
6. OwnPay team reviews your plugin
7. Published on marketplace

### Submission Requirements
- **Code quality** - Follow OwnPay coding standards
- **Documentation** - Clear README and usage instructions
- **Security audit** - No vulnerabilities or malicious code
- **Testing** - Compatible with current OwnPay versions
- **Support** - Provide issue support for users
