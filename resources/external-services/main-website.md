## Getting Started from Website

### For New Users
1. Visit [ownpay.org](https://ownpay.org)
2. Click "Get Started"
3. Choose:
   - **Try Demo** → [demo.ownpay.org](https://demo.ownpay.org)
   - **Install** → [Installation Guide](/getting-started/installation)
   - **Learn More** → This documentation

### For Developers
1. Visit [ownpay.org/developers](https://ownpay.org/developers)
2. Check out:
   - [Developer Guide](/developer/)
   - [API Documentation](https://docs.ownpay.org)
   - [Plugin Marketplace](https://plugins.ownpay.org)
3. Start building

### For Businesses
1. Visit [ownpay.org/enterprise](https://ownpay.org/enterprise)
2. Review:
   - Enterprise features
   - Support plans
   - Pricing
3. Contact sales
