This guide walks through creating your first OwnPay plugin.

## Step 1: Create plugin directory

```bash
mkdir -p modules/gateways/my-gateway/src
mkdir -p modules/gateways/my-gateway/resources/views
```

## Step 2: Create manifest.json

```json
{
  "slug": "my-gateway",
  "name": "My Custom Gateway",
  "version": "1.0.0",
  "type": "gateway",
  "entrypoint": "src/MyGateway.php",
  "namespace": "OwnPay\\Gateways\\MyGateway",
  "icon": "my-gateway.svg",
  "requires": {
    "ownpay": ">=0.2.0",
    "php": ">=8.3"
  }
}
```

## Step 3: Create the plugin class

```php
<?php
declare(strict_types=1);

namespace OwnPay\Gateways\MyGateway;

class MyGateway
{
    public function register(): void
    {
        // Register hooks, routes, events
        Hook::add('gateway.methods', [$this, 'addMethod']);
        Event::listen('payment.created', [$this, 'handlePayment']);
    }
    
    public function boot(): void
    {
        // Plugin initialization
    }
    
    public function uninstall(): void
    {
        // Cleanup on uninstall
    }
    
    public function addMethod(array $methods): array
    {
        $methods['my-gateway'] = [
            'name' => 'My Custom Gateway',
            'description' => 'Accept payments via My Gateway',
            'icon' => '/assets/img/gateways/my-gateway.svg',
        ];
        return $methods;
    }
    
    public function handlePayment(array $payment): void
    {
        // Process payment through your gateway
    }
}
```

## Step 4: Implement GatewayAdapterInterface

For gateway plugins, implement the adapter interface:

```php
class MyGateway implements GatewayAdapterInterface
{
    public function initiatePayment(array $data): array
    {
        // Call your gateway API
        // Return ['success' => true, 'checkout_url' => '...']
    }
    
    public function handleCallback(array $payload): array
    {
        // Process gateway callback
        // Return ['status' => 'completed', 'transaction_id' => '...']
    }
    
    public function refund(array $data): array
    {
        // Process refund through your gateway
    }
    
    public function supportedCurrencies(): array
    {
        return ['USD', 'EUR']; // Empty array = any currency
    }
}
```

## Step 5: Test the plugin

1. Enable the plugin via **System > Plugins**
2. Or via CLI: `php public/index.php plugin:enable my-gateway`
3. Test functionality
4. Verify hooks and events fire correctly

## Step 6: Add admin settings

```php
public function register(): void
{
    Menu::add('My Gateway', 'my-gateway', '/admin/gateways/configure/my-gateway');
    
    Route::get('/admin/gateways/configure/my-gateway', 'MyGatewayController@settings');
    Route::post('/admin/gateways/configure/my-gateway', 'MyGatewayController@save');
}
```

## Best practices

- Always declare `strict_types=1`
- Validate all input
- Use parameterized queries
- Escape output to prevent XSS
- Respect the permission system
- Store secrets in config, never hardcoded
- Log important events
- Write tests
