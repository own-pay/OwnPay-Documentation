## [0.1.0-beta] - 2026-06-18

This is the initial public beta release of the OwnPay core platform.

### Added
- **Multi-Brand Isolation**: Run multiple fully-branded checkout portals on custom white-label domains from a single server installation.
- **Android SMS Gateway**: Pair Android devices to capture, parse, and verify incoming transaction messages from banks.
- **Double-Entry Ledger Engine**: High-precision financial accounting built using PHP's `bcmath` extension.
- **Plugin Sandbox**: Extensible architecture supporting sandboxed gateway adapters and system addons.
- **Staff Control Directory**: Granular Role-Based Access Control (RBAC) supporting custom administrative permissions.
- **REST APIs & Webhooks**: Merchant-facing endpoints and outbound delivery handlers.
- **Self-Update Engine**: Integrated cryptographically signed core update checks.
